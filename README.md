## Tech Stack

- React
- TypeScript
- Vite
- WebSocket
- TailwindCSS

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/windate3411/order-book.git
```

2. Install dependencies

```bash
npm install
```

3. Run development server

```bash
npm run dev
```

4. Open http://localhost:5173 in your browser
